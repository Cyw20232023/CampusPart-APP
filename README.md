# 校兼帮原型系统自述说明

## 🎯 系统概述

校兼帮是一个面向校园的兼职招聘平台原型系统，支持学生和招聘者双身份切换，提供完整的岗位管理、实时沟通、技能认证等功能。

## 🏗️ 系统架构

### 技术栈
- **前端**: HTML5 + CSS3 + JavaScript (原生)
- **存储**: 浏览器 localStorage (本地数据持久化)
- **样式**: 响应式设计 + 移动端优先
- **交互**: 单页面应用 (SPA) 架构

### 核心功能模块

#### 1. 身份系统
- **双身份切换**: 学生/招聘者实时切换
- **登录方式**: 密码登录 + 短信验证码登录
- **记住密码**: 自动填充登录信息
- **测试账号**: 
  - 学生端: 小明 (密码任意)
  - 招聘端: 小红 (密码任意)

#### 2. 岗位管理
- **学生功能**:
  - 岗位浏览与筛选 (类型、薪资、地点)
  - 快速搜索与过滤
  - 一键报名申请
  - 保证金机制 (5%薪资)

- **招聘功能**:
  - 岗位发布与管理
  - 报名人员审核
  - 保证金设置
  - 岗位状态跟踪

#### 3. 实时沟通
- **消息中心**: 所有会话统一管理
- **一对一聊天**: 支持文本消息
- **系统通知**: 申请状态变更提醒
- **示例对话**: 3个预设沟通场景

#### 4. 技能认证
- **技能提交**: 学生上传技能证明
- **审核流程**: 招聘者审核技能认证
- **状态管理**: 待审核/已认证/已拒绝

#### 5. 简历管理
- **文件上传**: 支持PDF/图片/文档格式
- **本地存储**: 浏览器localStorage保存
- **预览功能**: 图片和PDF在线预览

### 示例数据初始化
系统预设了5个示例岗位：
1. 高等数学一对一家教 (200元/次)
2. 商场新品促销员 (300元/天) 
3. 学院论坛会务助理 (150元/次)
4. 运动会志愿者 (100元/天)
5. 软件测试兼职 (80元/任务)

### 业务流程

#### 学生端流程
1. 登录选择学生身份
2. 浏览筛选岗位
3. 上传简历(可选)
4. 点击报名申请
5. 等待审核通知
6. 与招聘者沟通
7. 完成兼职任务

#### 招聘端流程
1. 登录选择招聘身份
2. 发布岗位信息
3. 设置保证金
4. 审核报名人员
5. 沟通确认细节
6. 管理进行中岗位

## 📋 使用指南

### 快速开始
1. 打开 `校兼帮原型图.html`
2. 选择身份并登录
3. 根据身份体验相应功能

### 学生端操作
1. **浏览岗位**: 首页查看推荐岗位
2. **筛选搜索**: 使用搜索框和过滤器
3. **申请岗位**: 点击岗位卡片报名
4. **管理简历**: 在"我的"页面上传
5. **查看消息**: 底部导航进入消息中心

### 招聘端操作
1. **发布岗位**: 点击"发布"按钮
2. **审核申请**: 在岗位详情查看报名
3. **沟通确认**: 通过消息中心联系
4. **管理岗位**: 招聘页面查看所有发布

### 常用操作
- **身份切换**: 顶部身份按钮
- **退出登录**: "我的"页面底部
- **重置状态**: 调试功能(隐藏)
